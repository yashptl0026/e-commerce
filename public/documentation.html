Aetheria Luxe - React E-Commerce Theme

Premium Luxury Multi-Niche Storefront Template & Design System. Build high-converting e-commerce web applications with React, Vite, and Tailwind CSS.

Welcome

Thank you for purchasing Aetheria Luxe! Aetheria Luxe is an editorial-grade, ultra-premium React template built for high-end digital commerce. Featuring five pre-built storefront demos matching top-selling ThemeForest niches, dynamic transparent navigation menus, high-performance product Quick View HUD overlays, interactive Scandinavian living hotspot maps, and RTL direction mirror options, this template provides everything a developer needs to build stunning retail sites.

This template is built with 100% compliant React 19 standards, Vite for lightning-fast Hot Module Replacement (HMR), and Tailwind CSS v4 for utility-first styling.

Theme Features

Aetheria Luxe includes state-of-the-art interactive sections built specifically for premium marketplace shoppers:

  • 5 Distinct Storefront Demos:
    • Luxe Showcase (Default): High-end minimal multibrand store layout.
    • Haute Couture (Fashion): Editorial design focusing on large pictures, lookbooks, and fashion statements.
    • Cybertech Store (Electronics): Tech dark-mode grids, interactive specs tabs, and promotional drop timers.
    • Bespoke Timepieces (Watches): Rich typography, gold-accented mechanical watch grids, and brand heritage timelines.
    • Scandinavian Living (Furniture): Earthy minimalist design with an Interactive Room Hotspots Map.
  • Global Quick View Modal HUD: Instant overlays displaying rating stars, details, specs accordions, and quick checkout options.
  • Transparent Mega Menu: Multi-column navigation dropdown displaying category selectors, featured deals, and site demos.
  • Interactive Room Hotspots: Pulser pins positioned over room scenes that reveal products and launch Quick View buying panels.
  • Full LTR / RTL Direction Support: Switch page orientation instantly in the drawer switcher. Layouts dynamically mirror alignments.
  • Elegance Dark Mode Toggle: Transitions seamlessly between clean light styles and rich premium dark presets.
  • Automated Support Concierge Desk: Smart chatbot interface simulating live support tracking, return queries, and appointments.

Installation Guide

Follow these quick commands to set up the project files locally on your workstation:

Prerequisites

Ensure you have Node.js (Version 18+ recommended) and NPM installed on your machine.

Step-by-Step Setup

  1. Unzip the theme package on your hard drive.
  2. Open your command shell (Terminal or Command Prompt) and navigate to the directory: 
    cd /path/to/extracted/luxe
  3. Install project dependencies: 
    npm install
  4. Start the local development server: 
    npm run dev
  5. Open your browser and navigate to the local address displayed in your terminal (typically http://localhost:5173).

NPM Scripts

The theme provides pre-configured scripts in package.json for development, building, and validation:

Command Description
npm run dev Starts the Vite hot-reloading development server on localhost:5173.
npm run build Compiles TypeScript syntax and compiles the production bundles into the output folder /dist.
npm run lint Executes ESLint diagnostics to audit syntax code style compliance.
npm run preview Launches a local production server to test compiled builds before staging.

Typography & Fonts

We leverage Google Fonts to load rich modern typography families. These are configured dynamically in index.html and tokenized in CSS:

  • Display Font (Headings): "Plus Jakarta Sans", system-ui, sans-serif. It provides a geometric, high-fashion statement to headers, titles, and section markers.
  • Body Font (Paragraphs/Labels): "Inter", system-ui, sans-serif. Highly readable on both high-density digital displays and mobile viewports.
/* Configuration in src/index.css */
--font-display: "Plus Jakarta Sans", system-ui, sans-serif;
--font-body: "Inter", system-ui, sans-serif;

Glassmorphism Design System

Aetheria Luxe leverages sophisticated backdrop-filter glass layers to create premium depth cues. You can apply these styles via class helpers:

Class Name Styling Effect Best Use Cases

Customization Guide

Adding New Products

To inject products into your store, modify the mockProducts array located inside src/context/AppContext.tsx. Follow this TypeScript shape:

{
  id: 'unique-product-slug',
  name: 'Luxury Product Title',
  price: 999.00,
  category: 'fashion' | 'watches' | 'electronics' | 'decor' | 'beauty',
  subCategory: 'Bags',
  description: 'Detailed description string.',
  images: ['/image-path.jpg'],
  colors: [{ name: 'Black', hex: '#000000' }],
  sizes: ['S', 'M', 'L'],
  rating: 4.8,
  details: ['Detail bullet point 1', 'Detail bullet point 2']
}

Managing Categories

To add or update product categories, adjust the filtering controls on src/pages/Collection.tsx. Sidebar elements automatically sync brand filters by checking current products.
